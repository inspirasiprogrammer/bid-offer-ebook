# E-book Bid–Offer & Order Book

Selamat datang. E-book ini membahas cara membaca **bid–offer** dan **order book** untuk membantu pengambilan keputusan trading yang lebih presisi. Materi disusun bertahap dari dasar ke lanjut, dengan fokus pada praktik.

**Tujuan Pembaca Akhir:**
- Memahami peran bid, offer, spread, dan depth.
- Membaca intensi pasar dari antrian order book (likuiditas, ketidakseimbangan).
- Menyusun rencana eksekusi (entry/exit) berbasis konteks order book.
- Mengelola risiko dan psikologi saat eksekusi real-time.

> Catatan: E-book ini materi edukasi, bukan sinyal beli/jual. Risiko di pasar nyata tanggung jawab masing-masing.
